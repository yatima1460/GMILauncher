## How do I run the launcher?

### Windows

You can find Windows releases at:
<https://github.com/yatima1460/GMILauncher/releases>

### Mac

1. Download Whisky <https://getwhisky.app>
2. Create a new bottle with name "GMILauncher"
3. Wait until it get created (you should see a spinning wheel on the left)
4. Download the Windows release: <https://github.com/yatima1460/GMILauncher/releases>
5. Open right click the Launcher .exe with Whisky
   
Some games may show an error message like this one multiple times
![](docs/fake_error.png)

Click the various OKs and do not worry, the game should open fine

#### Run the launcher

1. Install Love2D into the Applications folder <https://love2d.org>
2. Download this source
3. In the source folder run it like this in the terminal `/Applications/love.app/Contents/MacOS/love .`

The launcher should launch fine.

#### Also run the games

1. If Mac ARM: install rosetta if not already `softwareupdate --install-rosetta`
1. Install brew <https://brew.sh/>
1. Install Wine `brew install wine-stable`
